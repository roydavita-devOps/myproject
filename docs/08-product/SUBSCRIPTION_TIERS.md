# Subscription Tiers

## Draft Tier Model

| Tier | Audience | Template Access | Notes |
| --- | --- | --- | --- |
| Basic | New local businesses | Standard templates | Lowest friction entry point. |
| Pro | Growing businesses | Standard + Premium templates | Main commercial plan. |
| Premium | Competitive businesses | Standard + Premium + Luxury + Exclusive templates | Highest value plan. |

## Access Rules

Future implementation should support:

```text
tenant
├── subscription_plan
└── allowed_template_access
```

Template access should not be hardcoded only to business type.

## Upgrade Drivers

- Better template quality.
- Premium industry layouts.
- More polished customer-facing presentation.
- Higher trust and conversion.
