# Template Marketplace

## Product Principle

Business Type is not Template.

```text
Business Type != Template
Business Type = Recommendation
Template = User Choice
```

Business Type should help UMKM Builder recommend a starting point. It should not permanently lock the tenant into a template.

## Examples

Valid:

```text
Business Type = Cafe
Template = Restaurant Luxury
```

Valid:

```text
Business Type = Clinic
Template = Corporate Executive
```

Valid:

```text
Business Type = Laundry
Template = Corporate Clean
```

## Future Features

- Template Catalog.
- Template Preview.
- Change Template.
- Recommended Templates.
- Premium Templates.
- Luxury Templates.
- Exclusive Industry Templates.

## Marketplace Model

The marketplace should allow users to:

- Browse templates.
- Preview templates with their own business data.
- See recommended templates based on business type.
- Select a different template manually.
- Upgrade plan to access premium templates.

## Template Categories

Potential categories:

- Restaurant.
- Cafe.
- Laundry.
- Clinic.
- Corporate.
- Retail.
- Local services.
- Premium landing pages.

## Access Model

Template access should be controlled by subscription plan.

| Access Level | Description |
| --- | --- |
| Standard | Included in Basic plan. |
| Premium | Included in Pro and Premium plans. |
| Luxury | Included in Premium plan. |
| Exclusive | High-value templates for specific industries or campaigns. |
